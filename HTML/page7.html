    • Ventana de Registro de Usuario

    Descripción General

    Esta ventana permite a los nuevos usuarios registrarse en la aplicación introduciendo sus datos personales. Una vez completada la información, los datos se guardan en la base de datos del sistema.

    Componentes de la Ventana

    1. Título Principal
      • Texto: Sing up!
      • Función: Indica al usuario que debe completar el registro para crear una cuenta.
    2. Campos de Entrada

      3. Los usuarios deben rellenar los siguientes campos con su información personal:

      • Mail: Dirección de correo electrónico del usuario.
      • Name: Nombre del usuario.
      • Last Names: Apellidos del usuario.
      • DNI: Número de Identificación del usuario.
      • Nickname: Alias o nombre de usuario que se utilizará en la aplicación.
      • Password: Contraseña de acceso a la cuenta.
    3. Botones Funcionales
      • Ok: Guarda la información introducida en la base de datos y finaliza el proceso de registro.
      • Cancel: Cancela el proceso de registro y cierra la ventana, sin guardar datos.

    Funcionamiento

    1. El usuario debe rellenar todos los campos de información.
    2. Al hacer clic en "Ok", el sistema valida los datos y los almacena en la base de datos.
    3. Si el usuario selecciona "Cancel", se descarta cualquier información introducida y se cierra la ventana.

    Resumen

    La ventana de Registro de Usuario permite a los nuevos usuarios ingresar su información básica y crear una cuenta en la aplicación. Esta funcionalidad es crucial para el acceso y gestión de perfiles dentro del sistema.
