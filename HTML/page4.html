    1. Ventana Principal de Gestión de Correos

      2. Descripción General

      Esta ventana permite al usuario gestionar los correos electrónicos dentro de su sistema de mensajería. Se ofrecen funcionalidades comunes para la navegación y organización de mensajes.

      Componentes de la Ventana

      1. Botón "Create New"

        2. Permite crear un nuevo correo electrónico. Al hacer clic, se abrirá la ventana de envío de correos.

        • Función: Redirige a la ventana de "Send Mail" (Envío de Correos).
      2. Carpetas de Correo

        3. Una lista de secciones de correos donde el usuario puede navegar entre los mensajes:

        • Inbox: Muestra los correos recibidos.
        • Unread: Filtra los mensajes no leídos.
        • Sent: Acceso a los correos enviados.
        • Draft: Muestra los borradores guardados.
        • Spam: Contiene los correos no deseados.
        • Trash: Almacena los mensajes eliminados.
      3. Área Principal de Visualización

        4. La sección grande de color gris permite mostrar la lista de correos electrónicos o los detalles del mensaje seleccionado.

      4. Botones Inferiores
        • Refresh: Actualiza la lista de correos electrónicos para mostrar los mensajes más recientes.
        • Back: Regresa a la ventana anterior o al menú principal.
    2. Ventana Principal de Gestión de Correos

      3. Descripción General

      Esta ventana permite redactar y enviar correos electrónicos. Incluye la funcionalidad de adjuntar archivos.

      Componentes de la Ventana

      1. Campos de Entrada
        • To: Campo donde el usuario debe introducir la dirección de correo del destinatario.
        • Topic: Aquí se escribe el asunto del correo.
        • Message: Un área de texto amplia donde el usuario redacta el cuerpo del correo.
      2. Botones Funcionales
        • Add File: Permite adjuntar archivos al correo electrónico. Al hacer clic, se abrirá un cuadro de diálogo para seleccionar el archivo.
        • Send: Envía el correo con la información ingresada.
        • Cancel: Cancela la operación y regresa a la ventana principal sin guardar cambios.
      3. Sección Inferior de Archivos Adjuntos

        4. Un espacio donde se mostrarán los archivos añadidos al correo electrónico.
